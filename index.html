hammer by jsgoecke

Adhearsion component to generate a flood of test calls to 'hammer' another telephony system

The Hammer is used to generate live inbound call load on an Asterisk or any other telephony system. The Hammer also provides a facility to play DTMF tones at the beginning of a call in order to traverse an IVR menu and reach various inbound call routes. The intended is use this facility in both engineering QA as well as a tool for operations in the field to load test each and every Asterisk or voice platform implementation.

Dependencies

  • Adhearsion v0.8.0+
  • CouchRest v0.10.1+ (for logging Events support)

Install

sudo gem install jsgoecke-hammer --source=http://gems.github.com/
